# webb-project--antonio
Detta projekt är en webbplats som ger information om stöd och hjälp för både unga och vuxna i olika livssituationer.

## Funktioner

- Tydlig navigationsmeny
- Sidor för stöd, rättigheter och kontakt
- Kontaktformulär med inputfält och knapp
- Responsiv design (fungerar på mobil och dator)

## Tekniker

- HTML (semantisk struktur: header, nav, main, section, article, footer)
- CSS
  - Flexbox (navigation)
  - Grid (kortlayout)
  - Media queries (responsivitet)
  - Hover- och focus-effekter

## Design

Designen är gjord för att kännas lugn och trygg:
- Mjuka färger
- Centrerad text
- Begränsad radlängd för bättre läsbarhet
- Tydlig call-to-action (knapp)
b##🧪 Felsökning

- Fixat problem med CSS-specificitet (t.ex. varningsruta)
- Löst buggar i hover-effekter (translateY)
- Justerat layout och spacing för bättre balans
